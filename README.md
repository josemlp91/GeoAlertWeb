GeoAlertWeb
===========

**Proyecto: Geo-Alert System :** Sistema de notificaciones basado en marcas (latitud y longitud) de posiciones geograficas y proximidad a etiqueta NFC.

El pocicionamiento se realiza usando la API de google Maps
